# Steganography

Written in C, this application is intended to hide text and images within another image.

## Requirements

* Linux Mint
* GCC version 5.4.0
* libpng (`sudo apt install zlib1g-dev libpng-dev`)

## To Run

Run `make` to compile, and `./hidden` to execute the file.
